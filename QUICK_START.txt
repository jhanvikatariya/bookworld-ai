## ğŸ”§ Customization

### Change Logo
Replace `static/images/logo.png` or `logo.svg` with your custom logo.

### Change Colors
Edit `static/css/style.css` at line 7-12:
```css
:root {
    --sage-green: #7c9885;    /* Change primary color */
    --warm-beige: #b8a08a;    /* Change secondary color */
    --soft-gold: #d4a574;     /* Change accent color */
    --off-white: #faf8f5;     /* Change background */
    --dark-green-gray: #2c3e3a; /* Change text color */
}
```

### Add/Remove Categories
Edit `static/js/config.js` around line 13:
```javascript
CATEGORIES: [
    { id: 'fiction', name: 'Fiction', icon: 'fa-book', query: 'subject:fiction' },
    // Add or remove categories here
]
```

### Change Port
Edit `app.py` last line:
```python
app.run(debug=True, host='0.0.0.0', port=5000)  # Change 5000 to your port
```
